### 🌐 **[ STEP 2 ] : THE SECRET DOOR IN BRAVE** 🟡🟢🔵

Open Brave. See that long bar at the top where you usually type *"how to tie my shoes"*? Type this in instead:
`brave://extensions`
Hit the big **Enter** button. If you hit it correctly, a page with lots of boxes will show up. 🗳️✨

### 💡 **[ STEP 3 ] : THE "I'M A HACKER" SWITCH** 🟢🔵🟣

### 📥 **[ STEP 4 ] : LOADING THE THINGY** 🔵🟣💗

Now that you’ve flipped the switch, click the button on the left that says **"Load unpacked"**.
A window will pop up. Find your `trustlayer-3` folder. Click it **once**. Click **"Select Folder"**.
If you did this right, a new box for **TrustLayer v3** will appear. If it didn’t, you probably clicked a single file like a toddler. Try again. 👶🚫

### 🔑 **[ STEP 5 ] : THE GEMINI API KEY (THE "GROWN-UP" PART)** 🟣💗✨

The extension needs **"brain juice"** from Google to work. 🧠🧃

1. Click the little **Puzzle Piece** 🧩 icon next to your search bar.

3. Click the **Google Gemini** button. 🔵
4. You need an API key. Go to [Google AI Studio](https://aistudio.google.com/app/apikey). Click the button that says "Create API key". 🔑

6. **Paste it** into the box in the TrustLayer popup.
7. Click **"Save Settings"**. If you get an error, you probably copied it wrong. Try to use your eyes this time. 👀🙄
